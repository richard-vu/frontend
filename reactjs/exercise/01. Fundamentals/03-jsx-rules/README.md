# JSX Rules Exercise / Bài tập về Quy tắc JSX

In this exercise, you will learn and apply the core rules of writing JSX in React.

_Trong bài tập này, bạn sẽ học và áp dụng các quy tắc cốt lõi của việc viết JSX trong React._

## Create a New Component / Tạo một Component mới

1. Create a new file called `JSXRules.jsx`.
   
   _Tạo một file mới có tên `JSXRules.jsx`._

2. Inside this file, write a functional component called `JSXRules`.
   
   _Bên trong file này, viết một functional component có tên `JSXRules`._

The component should return a `<div>` containing the following:

_Component này nên trả về một `<div>` chứa các nội dung sau:_

- A `<h1>` element with the text `"JSX Rules"`.
  
  _Một phần tử `<h1>` với nội dung là `"JSX Rules"`._

- A paragraph (`<p>`) that lists at least 3 rules of JSX:
  
  _Một đoạn văn (`<p>`) liệt kê ít nhất 3 quy tắc của JSX:_
  
  - JSX must return a **single parent element**.
    
    _JSX phải trả về một **phần tử cha duy nhất**._
    
  - JSX elements must be **properly closed**.
    
    _Các phần tử JSX phải được **đóng đúng cách**._
    
  - JSX attributes are written using **camelCase** (e.g., `className` instead of `class`).
    
    _Các thuộc tính JSX được viết theo kiểu **camelCase** (ví dụ: `className` thay vì `class`)._
