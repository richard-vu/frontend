# React Foundations / Nền tảng React

To effectively learn Next.js, it helps to be familiar with JavaScript, React, and related web development concepts. But JavaScript and React are vast topics. How do you know when you're ready to start using Next.js?

*Để học Next.js một cách hiệu quả, việc làm quen với JavaScript, React và các khái niệm phát triển web liên quan là rất hữu ích. Nhưng JavaScript và React là những chủ đề rộng lớn. Làm thế nào để bạn biết khi nào mình đã sẵn sàng bắt đầu sử dụng Next.js?*

Welcome to the React Foundations course! This beginner-friendly, example-led course will guide you through the prerequisite knowledge for Next.js. You will build a simple project step-by-step; starting with a JavaScript application, then migrating it to React and Next.js.

*Chào mừng bạn đến với khóa học Nền tảng React! Khóa học thân thiện với người mới bắt đầu và dựa trên ví dụ này sẽ hướng dẫn bạn qua những kiến thức tiên quyết cho Next.js. Bạn sẽ xây dựng một dự án đơn giản từng bước một; bắt đầu với một ứng dụng JavaScript, sau đó chuyển đổi nó sang React và Next.js.*

Each section builds on the previous one, so you can choose where to start depending on what you already know.

*Mỗi phần được xây dựng dựa trên phần trước đó, vì vậy bạn có thể chọn nơi bắt đầu tùy thuộc vào những gì bạn đã biết.*

## Prerequisite knowledge / Kiến thức tiên quyết

This tutorial assumes knowledge of HTML, CSS, JavaScript, and no knowledge of React. If you're already familiar with React, you can skip to the [From React to Next.js](https://nextjs.org/learn/react-foundations/from-react-to-nextjs) chapter or take [Dashboard App](https://nextjs.org/learn/dashboard-app) course.

*Hướng dẫn này giả định bạn đã có kiến thức về HTML, CSS, JavaScript và chưa có kiến thức về React. Nếu bạn đã quen thuộc với React, bạn có thể chuyển sang chương [Từ React đến Next.js](https://nextjs.org/learn/react-foundations/from-react-to-nextjs) hoặc tham gia khóa học [Dashboard App](https://nextjs.org/learn/dashboard-app).*

## System requirements / Yêu cầu hệ thống

Before you start this course, make sure you have the following installed:

*Trước khi bắt đầu khóa học này, hãy đảm bảo bạn đã cài đặt những thứ sau:*

- Node.js 20.12.0 or later installed.
  - *Node.js 20.12.0 hoặc phiên bản mới hơn.*
- Operating systems: macOS, Windows (including WSL), or Linux.
  - *Hệ điều hành: macOS, Windows (bao gồm WSL) hoặc Linux.*
- VSCode or another text editor of your choice.
  - *VSCode hoặc một trình soạn thảo văn bản khác mà bạn chọn.*
